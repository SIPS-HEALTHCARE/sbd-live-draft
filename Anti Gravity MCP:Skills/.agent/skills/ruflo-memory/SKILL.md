# /memory — Ruflo Vector Memory

This is a bridge command for the Ruflo framework. It provides access to the persistent vector memory and task history.

## Commands
- `/memory search --query "search term"`: Find semantic matches in history.
- `/memory store --key "key" --value "value"`: Store important context.

> [!NOTE]
> This command requires the Ruflo skill to be installed in the same vault.
