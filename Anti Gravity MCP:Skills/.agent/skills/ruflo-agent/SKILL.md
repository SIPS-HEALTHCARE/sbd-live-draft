# /agent — Ruflo Agent Management

This is a bridge command for the Ruflo framework. It allows you to spawn specialized agents for specific tasks.

## Commands
- `/agent spawn --type coder --name dev-1`: Add a coder.
- `/agent spawn --type reviewer --name reviewer-1`: Add a reviewer.
- `/agent list`: List all active agents.

> [!NOTE]
> This command requires the Ruflo skill to be installed in the same vault.
