# /swarm — Ruflo Swarm Orchestration

This is a bridge command for the Ruflo framework. It allows you to initialize and manage multi-agent swarms.

## Commands
- `/swarm init --topology hierarchical`: Coordinated team with a lead.
- `/swarm init --topology mesh`: Peer-to-peer network.
- `/swarm start --objective "Task"`: Launch the swarm.

> [!NOTE]
> This command requires the Ruflo skill to be installed in the same vault.
